### AMD CMD (异步加载模块)

> 现在很少使用

### Common.js(node.js)

> 模块加载 js 文件的过程是运行时加载的，并且是同步的

### ES Module

> 加载 js 文件的过程是编译（解析）时加载的，并且是异步的

```html
<script src="01.js" type="module"></script>
<script src="02.js" type="module"></script>
```

```node
/**
 *01.js
 */

let name = "liou";
let say = (name) => {
  console.log("this is " + name);
};
let age = 18;
export { name, say }; //注意这里的{}不是对象
export default age;

/**
 *02.js
 */

import { name, say } from "./01.js";
console.log(name); //liou
```
